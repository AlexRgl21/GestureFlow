# GestureFlow

GestureFlow es una aplicación web que permite controlar la reproducción
de contenido multimedia mediante gestos de la mano detectados por la cámara.

El sistema utiliza visión por computadora para interpretar gestos simples
y ejecutar acciones dentro de un reproductor multimedia integrado.

## Funcionalidades

- Detección de gestos en tiempo real
- Control del reproductor de video
- Interfaz web simple
- Uso de cámara web

## Gestos soportados

✊ Puño → Play / Pause  
✋ Mano abierta → Siguiente  
👉 Apuntar → Anterior  

## Requisitos

- Navegador web moderno
- Cámara web
- Permisos de cámara habilitados

## Tecnologías

- HTML
- CSS
- JavaScript
- MediaPipe
- TensorFlow.js
