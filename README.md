# GestureFlow

GestureFlow es una aplicación web que permite controlar la reproducción de contenido multimedia mediante gestos de la mano detectados por la cámara.

El sistema utiliza visión por computadora para interpretar gestos simples y ejecutar acciones dentro de un reproductor multimedia integrado.

---

## Funcionalidades

- Detección de gestos en tiempo real mediante cámara web
- Control de reproducción multimedia usando gestos
- Configuración personalizable de acciones para cada gesto
- Reproductor multimedia integrado en la aplicación
- Interfaz web simple e intuitiva

---

## Gestos soportados

GestureFlow reconoce tres gestos principales.

Las acciones asociadas a cada gesto pueden ser **configuradas por el usuario** desde el menú de configuración dentro de la aplicación.

✊ **Puño**

✋ **Mano abierta**

👉 **Apuntar**

---

## Configuración de gestos

El usuario puede personalizar la acción asociada a cada gesto desde el menú de configuración.

Esto permite adaptar los gestos para controlar diferentes funciones del reproductor multimedia según las preferencias del usuario.
